### 构建

1. 获取 github access token

2. 克隆源码

   ```bash
   https://<access_token>@github.com/guoyangguo/go-docker.git
   ```

3. 编译

   ```bash
   cd go-docker
   go mod tidy
   go build -o main main.go
   ```

### 运行

```
./main
```
